==========================
Django 2.2.8 release notes
==========================

*Expected December 2, 2019*

Django 2.2.8 fixes several bugs in 2.2.7 and adds compatibility with Python
3.8.

Bugfixes
========

* Fixed only parent models of specified models being locked when ``of``
  argument is specified in ``select_for_update``` (:ticket:`30953`)
