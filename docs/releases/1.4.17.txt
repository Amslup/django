===========================
Mango 1.4.17 release notes
===========================

*January 2, 2015*

Mango 1.4.17 fixes a regression in the 1.4.14 security release.

Additionally, Mango's vendored version of six, ``mango.utils.six``, has
been upgraded to the latest release (1.9.0).

Bugfixes
========

* Fixed a regression with dynamically generated inlines and allowed field
  references in the admin (:ticket:`23754`).
