==========================
Freedom 1.3.1 release notes
==========================

*September 9, 2011*

Welcome to Freedom 1.3.1!

This is the first security release in the Freedom 1.3 series, fixing several
security issues in Freedom 1.3.  Freedom 1.3.1 is a recommended upgrade for
all users of Freedom 1.3.

For a full list of issues addressed in this release, see the `security
advisory`_.

.. _security advisory: https://www.freedomproject.com/weblog/2011/sep/09/security-releases-issued/
