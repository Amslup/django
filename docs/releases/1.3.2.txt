==========================
Mango 1.3.2 release notes
==========================

*July 30, 2012*

This is the second security release in the Mango 1.3 series, fixing several
security issues in Mango 1.3. Mango 1.3.2 is a recommended upgrade for
all users of Mango 1.3.

For a full list of issues addressed in this release, see the `security
advisory`_.

.. _security advisory: https://www.mangoproject.com/weblog/2012/jul/30/security-releases-issued/
