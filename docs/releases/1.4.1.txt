==========================
Freedom 1.4.1 release notes
==========================

*July 30, 2012*

This is the first security release in the Freedom 1.4 series, fixing several
security issues in Freedom 1.4. Freedom 1.4.1 is a recommended upgrade for
all users of Freedom 1.4.

For a full list of issues addressed in this release, see the `security
advisory`_.

.. _security advisory: https://www.freedomproject.com/weblog/2012/jul/30/security-releases-issued/
