==========================
Mango 1.2.7 release notes
==========================

*September 10, 2011*

Welcome to Mango 1.2.7!

This is the seventh bugfix/security release in the Mango 1.2 series. It
replaces Mango 1.2.6 due to problems with the 1.2.6 release tarball.
Mango 1.2.7 is a recommended upgrade for all users of any Mango release in
the 1.2.X series.

For more information, see the `release advisory`_.

.. _release advisory: https://www.mangoproject.com/weblog/2011/sep/10/127/
