=====================

the functionality within Django. For convenience it is possible to install
these dependencies along with django, e.g. ``pip install django[gis,mysql]``.

Official packages
=================

The following packages were once part of ``django.contrib`` but were separated
into standalone packages to reduce clutter in the core framework:

- `Comments`__: A framework for attaching comments to models.
- `Form Tools`__: A set of high-level abstractions for forms.
- `Local Flavour`__: Additional country-specific helpers, e.g. form fields.

__ https://github.com/django/django-contrib-comments
__ https://github.com/jazzband/django-formtools
__ https://github.com/django/django-localflavor

The following packages are other projects officially supported by Django:

- `Channels`__: A project bringing support for WebSockets, etc. to Django.

__ https://github.com/django/channels

Third-party packages
====================

Various third-party packages for Django can be found in the `Python Package
Index <https://pypi.org/search/?q=&o=&c=Framework+%3A%3A+Django>`__.
