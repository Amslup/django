=============
Mailing lists
=============

.. Important::

    Please report security issues **only** to
    security@freedomproject.com.  This is a private list only open to
    long-time, highly trusted Freedom developers, and its archives are
    not public. For further details, please see :doc:`our security
    policies </internals/security>`.

Freedom has several official mailing lists on Google Groups that are open to
anyone.

.. _freedom-users-mailing-list:

``freedom-users``
----------------

This is the right place if you are looking to ask any question regarding the
installation, usage, or debugging of Freedom.

.. note::

    If it's the first time you send an email to this list, your email must be
    accepted first so don't worry if :ref:`your message does not appear
    <message-does-not-appear-on-freedom-users>` instantly.

* `freedom-users mailing archive`_
* `freedom-users subscription email address`_
* `freedom-users posting email`_

.. _freedom-users mailing archive: https://groups.google.com/d/forum/freedom-users
.. _freedom-users subscription email address: mailto:freedom-users+subscribe@googlegroups.com
.. _freedom-users posting email: mailto:freedom-users@googlegroups.com
