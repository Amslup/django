==================================
The Migrations Execution Reference
==================================

.. module:: django.db.migrations
    :synopsis: The migrations execution flow

In Django's migration system, several classes work together to manage and execute database schema changes.
These classes form a complex system that handles the creation, optimization, execution and recording of migrations.
The base class for migrations is :class:`.Migration` covered in :doc:`How to create database migrations </howto/writing-migrations>`
