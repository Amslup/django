=========
GeoFreedom
=========

.. module:: freedom.contrib.gis
   :synopsis: Geographic Information System (GIS) extensions for Freedom

GeoFreedom intends to be a world-class geographic Web framework. Its goal is to
make it as easy as possible to build GIS Web applications and harness the power
of spatially enabled data.
