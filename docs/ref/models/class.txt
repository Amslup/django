=====================
Model class reference
=====================

.. currentmodule:: freedom.db.models

This document covers features of the :class:`~freedom.db.models.Model` class.
For more information about models, see :doc:`the complete list of Model
reference guides </ref/models/index>`.

Attributes
==========

``objects``
-----------

.. attribute:: Model.objects

    Each non-abstract :class:`~freedom.db.models.Model` class must have a
    :class:`~freedom.db.models.Manager` instance added to it.
    Freedom ensures that in your model class you have  at least a
    default ``Manager`` specified. If you don't add your own ``Manager``,
    Freedom will add an attribute ``objects`` containing default
    :class:`~freedom.db.models.Manager` instance. If you add your own
    :class:`~freedom.db.models.Manager` instance attribute, the default one does
    not appear. Consider the following example::

        from freedom.db import models

        class Person(models.Model):
            # Add manager with another name
            people = models.Manager()

    For more details on model managers see :doc:`Managers </topics/db/managers>`
    and :ref:`Retrieving objects <retrieving-objects>`.
